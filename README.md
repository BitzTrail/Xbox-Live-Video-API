# XboxReplay - Sample Project

<img src="sample-project.png" width="720">

### Installation
```
$ git clone https://github.com/XboxReplay/sample-project.git
$ cd sample-project
$ npm install
```

### Start
```
$ XBL_EMAIL=user@live.com \
  XBL_PASSWORD=password \
  npm start
```
